Installation

Requirements:

  • Go 1.19 or higher
  • Ollama (optional, for functional testing)

Install from source:

# download dependencies
go mod download && go mod tidy

# build
go build -v .

Run directly for development:

go run main.go

If you want to serve docs locally during writing:

pip install mkdocs
mkdocs serve
