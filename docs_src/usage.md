# Usage

This section explains common usage patterns for `kirk-ai`.

## List models

```bash
./kirk-ai models
```

## Chat

```bash
./kirk-ai chat "Hello! Tell me a joke"
```

## Embeddings

```bash
./kirk-ai embed "This is some text to embed"
```

## Build and run

```bash
go build -v .
./kirk-ai --help
```
