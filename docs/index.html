kirk-ai

Lightweight CLI for interacting with Ollama AI models.

Overview

This repository provides a command-line tool called kirk-ai to interact with Ollama models for chat, code generation, translation, embeddings, and benchmarking.

Docs & Usage

Build the binary locally and run the commands from the repository root. See README.md for full usage examples.

Quick commands: 

# download dependencies
go mod download && go mod tidy

# build
go build -v .

# show help
./kirk-ai --help

GitHub Pages

If the site doesn't appear automatically, go to your repository Settings → Pages and ensure the Source is set to "Deploy from a GitHub Action" or the "docs/" folder on the main branch.
